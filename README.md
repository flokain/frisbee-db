# Test

To start the dev server for the UI, use:
```
> gradlew :frontend:startUi
```

# Develop

## UI

UI development is done using Angular + Yarn
```
> cd frontend
> yarn install
> yarn start
```

# Build
Create the ZIP
```
> gradlew createZip
```
